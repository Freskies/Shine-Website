# Shine Parkour Website

This is the website for "Shine &mdash; School of Movement."

The site should be completely accessible in Italian and in English. Mainly it should be Italian, but
it should be possible to switch between the two languages by the browser language settings.

The site should be responsive and optimized for mobile devices. It should have a clean and modern design, with a focus
on usability and accessibility. The content should be organized in a logical and easy-to-navigate manner,
with clear headings and subheadings. The site should be optimized for search engines
and should include relevant keywords and meta-descriptions.

This is a page for Shine asd. The main purpose is to allow user to find information for courses
and a cta to book a try lesson.

The subgoals are:

- tell the story of the association
- it tells the story of parkour
- explain why you should do parkour
- a spot map with all places to do parkour (a person who travels needs that)
- a page with all the team (and a separate page for each one with a description
  of their history with pk formation, shine and parkour in general)
- Promote our gym and explain that we use that, but mainly train outside
- Give info for everyone who wants to hire us to do a show or an event
